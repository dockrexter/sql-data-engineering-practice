# SQL Practice Notebooks

This repository contains hands-on SQL practice notebooks focused on building strong fundamentals in relational databases and query design. The notebooks demonstrate practical usage of SQL for data exploration, transformation, and analysis.

## Contents

- SQL query examples using `SELECT`, `WHERE`, `ORDER BY`
- Aggregate functions and grouping
- Joins across multiple tables
- Subqueries and nested queries
- Data filtering and logical conditions
- Practice-oriented problem solving

## Tools & Technologies

- SQL
- Jupyter Notebook
- Relational database concepts

## Purpose

The goal of this repository is to document consistent SQL practice and reinforce core database concepts through hands-on examples. It serves as a personal reference as well as a demonstration of practical SQL skills.

## How to Use

1. Clone the repository
2. Open the notebook using Jupyter Notebook or JupyterLab
3. Run the SQL queries and explore the examples
4. Modify queries to experiment with different scenarios
